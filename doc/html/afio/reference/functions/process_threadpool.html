process_threadpool

Returns the process threadpool.

Description

On first use, this instantiates a default std_thread_pool running BOOST\u005fAFIO\u005fMAX\u005fNON\u005fASYNC\u005fQUEUE\u005fDEPTH threads which will remain until its shared count reaches zero.
