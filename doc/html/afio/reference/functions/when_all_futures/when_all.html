when_all (iterator batch of ops exception propagating)

Makes a future vector of results from all the supplied futures.

Description

Returns a result when all the supplied ops complete. Propagates exception states.

This is an implementation of WG21 N3634's when_all() which uses process_threadpool() to allow you to create a future which only becomes available when all the supplied futures become available.
